Comparison Operators
Operators Example code
&& Logical AND

code will be run only when both condition ( condition 1 && condition 2 ) will must be True

 let age = 20;
if ( age >= 18 && age <= 21 ) {
console.log ('you are eligible') ;
}
Ans = you are eligible

In above example both condition are true, because age is greater then 18 (condition 1st ) and age is Less then 21
|| Logical OR

code will be run only when any ONE condition ( condition 1 || condition 2 ) will must be True

 let age = 20;
if ( age >= 18 || age <= 10 ) {
console.log ('you are eligible') ;
}
Ans = you are eligible

In above example condition 1st is true. Because age is greater then 18, But 2nd condition is false bcoz age is not Less then 10
! Logical NOT

if your condition is True then Logical Operator will answer you in reverse order

 let age = 100;
console.log(! age <= 200);
Ans = true
